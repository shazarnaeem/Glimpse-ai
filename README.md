# Glimpse AI

A modern AI chat interface with a sleek design and powerful features.

## Features

- Modern, responsive UI
- Dark/Light theme support
- Code highlighting
- Chat history
- Share functionality
- PDF and text export
- Email sharing

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Particles.js
- Highlight.js
- jsPDF
- Font Awesome

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start chatting!
